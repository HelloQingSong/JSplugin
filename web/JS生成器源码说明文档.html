一、简介

源码库 : https://github.com/HelloQingSong/JSplugin.git
目前，js 生成器可以生成的脚本分为五类(其中，每一类脚本又可以根据横向表格与纵向表格分成两类)

  • 实时核查 – 数值范围检测；
  • 光标跳转
  • radio撤销
  • 设置文本框格式
  • 大小写转换
    并且这些功能，可以根据需要相互组合（注意：部分功能在逻辑上存在冲突，未作检测，如: radio 撤销与实时核查)

二、横向表格

1. 页面-后端 信息交互参数

  • jsId —— 定位控件id后缀 – js 脚本id
  • functionsNeeded —— 用户选择的脚本功能
    1. RealTimeVer —— 实时核查 – 数值范围检测
      参数:
      • upperBound : 上界 ==> 数字
      • lowerBound : 下界 ==> 数字
    2. FocusTransfer —— 光标跳转
      参数:
      • maxLength ：可以输入的最大字符个数 ==> 整数
      • activation ： 是否激活光标跳转功能 ==> true or false
      • nextControlWay : 选择以那种方式设置下一个控件定位脚本 default(默认) or customize(自定义)
      • nextControl ： 下一个控件定位脚本 ==> 自定义脚本
    3. RadioRevoked —— radio 选择撤销
      • 参数 : 无
    4. ChangeStyle —— 文本框样式
      参数 :
      • stylesNeedName ： ‘styleName’ ==> 样式名
      • stylesNeedValue : ‘参数’ ==> 样式值
    5. ToUpperCase—— 强制转换大小写
      参数：
      • toUpper: true==>默认，小写转大写；false ==> 大写转小写
  • locateFunction : 定位目标控件方式 ==> default(默认) or customize(定制)
  • locateExternalControl : 定位外部控件 ==> 自定义脚本
  • locateInternalControl : 定位内部控件 ==> 自定义脚本

2. 横向表格功能扩展说明

当需要扩展一个新的功能时，一般来说，需要修改(修改区域，已在图中注明)

  • CreateHorTableJs.jsp
  • CreateHorTableJsController.java

其中，额外添加的html 控件(如 : <div> 、<span>、<img>) 均通过 horizontal.properties配置；在horizontal.properties中，还包括其他与生成横向表格脚本相关的参数，如:

  • 定位控件
  • 目标控件定位脚本
    1. 定位外部控件
    2. 定位内部控件( 即 目标控件)

如果需要添加 html 控件，需要添加两个参数

  1. html 标签名 = <html 标签> </html 标签>
  2. {html 标签名}+BaseID=additon{html标签名}
